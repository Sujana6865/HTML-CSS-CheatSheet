Div, Section and Article

Which to use when? We often time get confused while dealing with this tags. Below we will see the difference between these tags:

  • <div> - most general purpose element.
  • <section> - slightly more specific than a div.
  • <article> - even more specific than a section.

HTML Table

HTML <table> tag represents tabular data. It is comprised of rows and columns.

Tag Name Description
<table> Table The wrapper element of all HTML table.
<thead> Table Head The head of a table with information about table column.
<tbody> Table Body The body of the table's main data.
<tr> Table Row The row of cells in a table.
<td> Table Data The cell of table that contains data.
<tfoot> Table Foot The summary of column.

Content

  • Content: Content of the box, where text and image appears.
  • Padding: Clears an area around the content. It is transparent.
  • Border: It goes around the padding and content.
  • Margin: Clears an area outside the border. It is also transparent.
